## How to Watch Amigoscode Videos
1.  Don't just "watch":
    Create and open *VideoNotes.java* in Cursor while watching
    Enter the useful example codes into the file

2.  Pause every 20 - 30 minutes and ask yourself:
    "Can I write the code I just saw without watching the video?"
    If not, rewatch that segment

3.  For each new concept in the video, type in the Cursor Chat:
    "Amigoscode just demonstrated [new concept], 
     please provide a different example to deepen my understanding."
